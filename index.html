Los diferentes tipos de errores dentro de un algoritmo
La impresión: Hay diferentes tipos de errores de impresión unos de esos errores es al escribir mal el comando por ejemplo: escribir mayor dos veces cuando lo correcto seria escribir el signo menor dos veces, poner comillas simples cuando lo correcto seria poner comillas dobles, etc.
La petición: Es guardar en una variable la petición por ejemplo: un error es no crear una variable, si no crea la variable no hay lugar donde guardar la petición, un error es escribir mal el comando ( poner menor dos veces cuando lo correcto es poner mayor dos veces)
La Declaración: Es la linea de toda las variables, por ejemplo: si se coloca comillas simples esta mal lo correcto es poner comillas dobles
La modificación: Es cambiarle el valor que se le dio al nombre de la variable, ejemplo: un error que se comete es colocar la variable despues de que ya haya pedido la petición lo correcto seria colocar la variable antes de que se pieda la petición.
